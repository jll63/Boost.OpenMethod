
## virtual_ptr_traits<Class, Policy>

:idprefix: virtual_ptr_traits_

### Description

`virtual_ptr_traits` is a class template used by `virtual_ptr` to manipulate the
pointer to the object.

### Synopsis

`virtual_ptr_traits` is defined in `<boost/openmethod/core.hpp>`.

```c++
namespace boost::openmethod {

template<class Class, class Policy>
struct virtual_ptr_traits {
    static bool constexpr is_smart_ptr = ...;

    using element_type = ...;

    static auto dynamic_type(const Class& obj) -> type_id;

    template<typename Other>
    static auto cast(const virtual_ptr<Class, Policy>& ptr)
        -> virtual_ptr<Other, Policy>;
};

} // namespace boost::openmethod
```

### Members

#### is_smart_ptr

```c++
static bool constexpr is_smart_ptr = ...;
```

`true` if the pointer to the object is a smart pointer.

#### element_type

```c++
using element_type = ...;
```

The type of the object pointed to. If `is_smart_ptr` is `false`, it is `Class`;
otherwise, it is the class of the object pointed to.

#### dynamic_type

```c++
static auto dynamic_type(const Class& obj) -> type_id;
```

Returns the _dynamic_ type_id of the object.

#### cast

```c++
template<typename Other>
static auto cast(const virtual_ptr<Class, Policy>& ptr)
    -> virtual_ptr<Other, Policy>;
```

Casts `ptr` to `Other`. If `Class` is a smart pointer, `Other` must be a smart
pointer of the same kind (e.g. `std::shared_ptr<Animal>` and
`std::shared_ptr<Cat>`).

### Specializations

#### virtual_ptr_traits<std::shared_ptr<Class>, Policy>

Defined in `<boost/openmethod/virtual_shared_ptr.hpp>`.

[cols="1,1"]
|===
| `is_smart_ptr` | `true`
| `element_type` | `Class`
| `dynamic_type(const ... & ptr)` | `Policy::dynamic_type(*ptr)`
| `cast(const ... & ptr)` | uses `std::static_pointer_cast` if possible, otherise uses `std::dynamic_pointer_cast`
|===

where `...` is `virtual_ptr<std::shared_ptr<Class>, Policy>`.

#### virtual_ptr_traits<std::unique_ptr<Class>, Policy>

Defined in `<boost/openmethod/virtual_unique_ptr.hpp>`.

[cols="1,1"]
|===
| `is_smart_ptr` | `true`
| `element_type` | `Class`
| `dynamic_type(const ... & ptr)` | `Policy::dynamic_type(*ptr)`
| `cast(... ptr)` | uses `static_cast` if possible, otherwise uses `dynamic_cast`
|===

where `...` is `virtual_ptr<std::unique_ptr<Other>, Policy>`.

NOTE: `cast` transfers the ownership of the object to the returned pointer.
