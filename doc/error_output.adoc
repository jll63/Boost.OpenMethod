
## error_output

### Synopsis

```c++
namespace boost {
namespace openmethod {
namespace policies {

struct error_output {};

}}}
```

### Description

`error_output` is a facet that provides a stream for writing error messages.

Implementations of `error_output` provide the following functions:

[cols="a,a", options="header"]

|===

| Name
| Description

| error_stream
| A static variable that satisfies the requirements of `RestrictedOutputStream`.

|===
