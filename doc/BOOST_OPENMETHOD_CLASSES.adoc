
[#BOOST_OPENMETHOD_CLASSES]

## BOOST_OPENMETHOD_CLASSES(CLASSES...[, POLICY])

### Synopsis

Defined in <boost/openmethod/macros.hpp>.

```c++
BOOST_OPENMETHOD_CLASSES(CLASSES...[, POLICY]);
```

### Description

Register `CLASSES` in POLICY. If `POLICY` is not specified, use the default
policy.

NOTE: The default policy is the value of `BOOST_OPENMETHOD_DEFAULT_POLICY` at
the point `<boost/openmethod/core.hpp>` is included. Changing the value of this
symbol has no effect after that point.

This macro is a wrapper around `use_classes`; see its documentation for more
details.
