

## virtual_<T>

### Description

Marks a formal parameter of a method as virtual. Requires a specialization of
`virtual_traits` for `T` and the `Policy` of the method. Specializations for
`T&`, `T&&`, `std::shared_ptr<T>` and `const std::shared_ptr<T>&` are provided.
See the documentation of `virtual_traits` for more information.

### Synopsis

```c++
namespace boost::openmethod {

template<typename T>
struct virtual_;

}
```
